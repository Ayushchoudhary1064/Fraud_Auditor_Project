FRAUD AUDITOR: A Visual Analytics Approach for Collusive Fraud in Health Insurance

This project aims to detect collusive fraud among health insurance claims using co-visit networks and weighted community detection.

Users can submit claims while the admin evaluates them using visual insights and analytics.

The system enhances transparency and automates fraud identification to assist insurers in decision-making.
